This scripts are part of TBFY project. TBFY project is divided into three layers:
(*) tbfy.data
(*) tbfy.analysis
(*) tbfy.web

Every TBFY layer has a more detailed description available in its own README.xxx.txt located in its base directory. This README.web.txt file is describing the role of "tbfy.web" scripts.

The role of this layer, a tbfy.web layer, is to serve end users analysis results in a human readable format. This is done through http protocol in a form of webpage.
